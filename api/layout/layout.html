布局

说明:

五方布局（layout）提供五个区域：east、west、north、south、center。以下是一些通常用法： north 区域可以用来显示网站的标语，south 区域可以用来显示版权以及一些说明，west 区域可以用来显示导航菜单， east 区域可以用来显示一些推广的项目，center 区域可以用来显示主要的内容。

如：

折叠状态显示标题>data-options="showCollapsedTitle:true"
底部
自动收起>data-options="collapsed:true"
菜单
主要的内容 
<div id="cc" class="hisui-layout" style="width:600px;height:400px;" data-options="clickExpand:true">   
	<div data-options="region:'north',title:'North',split:true,showCollapsedTitle:true,iconCls:'icon-w-list'" style="height:100px;padding:10px;">
		折叠状态显示标题><code>data-options="showCollapsedTitle:true"</code>
	</div>  
	<div data-options="region:'south',title:'',split:true" style="height:100px;padding:10px;">
		底部
	</div>
	<div data-options="region:'east',title:'East',iconCls:'icon-w-list',split:true,split:true,collapsed:true" 
		style="width:200px;padding:10px;">
			自动收起><code>data-options="collapsed:true"</code>
	</div>   
	<div data-options="region:'west',title:'菜单',iconCls:'icon-w-list',split:true,showCollapsedTitle:true" style="width:100px;padding:10px;">菜单</div>   
	<div data-options="region:'center',title:'',split:true" 
		style="padding:10px;">
		主要的内容
	</div>   
</div>

布局组件的相关配置

属性 说明 默认值
clickExpand 点击展开与否。为true时，展开区域不再悬浮 false

区域面板配置

属性 说明 默认值
showCollapsedTitle 在区域面板处于折叠状态的时候是否显示title false
