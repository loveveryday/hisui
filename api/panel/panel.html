面板

说明:

面板是其他组件或元素的容器

一、卡片式面板

卡片面板headerCls:'panel-header-card' 
<div class="hisui-panel" title="登记（不常用）" style="height:150px;width:400px;padding:20px" data-options="headerCls:'panel-header-card'">卡片面板</div>
灰色卡片面板"headerCls:'panel-header-card-gray'" 
<div class="hisui-panel" title="登记（不常用）" style="width:400px;height:150px;padding:20px" data-options="headerCls:'panel-header-card-gray'">灰色卡片面板</div>

二、灰色系面板

灰色面板只需修改headerCls:'panel-header-gray' 
<div class="hisui-panel" title="病人信息" style="width:400px;padding:20px" data-options="headerCls:'panel-header-gray',iconCls:'icon-paper',closable:true,collapsible:true,minimizable:true,maximizable:true"></div>
没有标题的灰色面板，只需修改bodyCls:'panel-body-gray',增加于2018-09-17 
<div class="hisui-panel panel-header-gray" title="" data-options="..."></div>
大一点面板 headerCls:'panel-header-big panel-header-gray' 
<div data-options="headerCls:'panel-header-big panel-header-gray',..."></div>
还可通过js生成 
<div id="accPanel" title="病人信息" style="width:400px;padding:20px;">还可通过js生成</div>
	<script type="text/javascript">
	var init = function () {
		var valbox = $HUI.panel("#accPanel",{
			iconCls:'icon-paper',
			closable:true,
			onBeforeClose:function(){alert("你点击了关闭!");return false},
			collapsible:true,
			onBeforeCollapse:function(){alert("你点击了收起!");return false},
			minimizable:true,
			onMinimize:function(){alert("你点击了最小化!");return false},
			maximizable:true,
			onMaximize:function(){alert("你点击了最大化!");return false},
			headerCls:'panel-header-gray'
		});
	}
	$(init);
	</script>

三、其它色系

白色面板只需修改headerCls:'panel-header-white'
没标题 没有title
属性 说明 默认值
headerCls 面板头样式类 null,表示蓝色 可选值：'panel-header-gray','panel-header-blue', 'panel-header-acc', 'panel-header-white','panel-header-card','panel-header-card-gray','panel-header-big'
bodyCls 面板内容样式类 null 可选值：'panel-body-gray'
