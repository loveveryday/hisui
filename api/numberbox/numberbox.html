数字验证框

说明:

数字验证框（numberbox）用来验证数字表单，在输入框上加入简单的验证 。

如：

数量
数字 值即时变化 
<div class="hisui-panel" title="登记" style="width:400px;padding:10px 30px 20px 30px" data-options="iconCls:'icon-save',
closable:true,collapsible:true,minimizable:true,maximizable:true">
	<table cellpadding="5">
		</tr>
			<td>数量</td>
			<td>
				<input class="hisui-numberbox textbox">
			</td>
		</tr>
		<tr>
			<td class="r-label">数字
			<td>
				<input class="hisui-numberbox textbox" data-options="isKeyupChange:true" id="nb">
			</td>
		</tr>		
	</table>
</div>
<script type="text/javascript">
	$(function(){
		$("#nb").on('keyup',function(e){
			console.log("numberbox value = "+$(this).numberbox('getValue'));
			if (e.keyCode==13){
				console.log("numberbox value enterkey ="+$(this).numberbox('getValue'));
			}
		});
	})
</script>
属性 说明 默认值
isKeyupChange 默认值false 值为false时blur时改变组件的值。为true时则keyup时改变组件的值2019-4-18
