纵向节点图组件(vstep)

说明：

可用于步骤图，流程图，闭环图等，可自定义节点后描述内容及格式。2021-02-26。

一.步骤图

上一步 下一步
配置项 说明 默认值 其它
currentInd 当前所在节点 1
stepHeight 单步高度 100
items 具体节点配置 Object { title:'title', context:"html/text" }
方法名 说明 入参 返回值
setStep 设置当前流程处于哪个节点 ind
getStep 获得当前流程处于哪个节点
getStepInd 获得当前流程处于的索引位置
prevStep 上一节点 无
nextStep 下一节点
事件名 说明 入参 类型
onSelect 当前选中对象 ind,item
