日期框

说明:

日期框（datebox）把可编辑的文本框和下拉日历面板结合起来，用户可以从下拉日历面板中选择日期，在文本框中输入的字符串可转换为有效日期。支持快捷t方式输入,t+2表示二天后,t-2表示二天前的日期。

如：

统计月份 获取值
生日 获取值
结算时间
Selected Date: 
<style>td{padding:5px 0;}</style>
<table> 
	<tr>
		<td class='r-label'>统计月份</td>
		<td class='r-label'>
			<input id="samth" class="hisui-datebox textbox" 
			data-options='onSelect:onSelect,formatter:function(date){var y = date.getFullYear(); var m = date.getMonth()+1;return y+"-"+m;},validParams:"YM"'/>
		</td>
		<td class='r-label'>
			<a class="hisui-linkbutton" id="samthBtn">获取值</a>
		</td>
	</tr>
	<tr>
		<td class='r-label'>生日</td>
		<td>
			<input id="db" class="hisui-datebox textbox" data-options='onSelect:onSelect'/>
		</td>
		<td class='r-label'>
			<a class="hisui-linkbutton" id="dbBtn">获取值</a>
		</td>
	</tr>
	<tr>
		<td class='r-label'>结算时间</td>
		<td>
			<input class="hisui-datetimebox textbox" data-options="onSelect:onSelect"/>
		</td>
	</tr>
</table>
<div style="margin:10px 0">
	<span>Selected Date: </span>
	<span id="result"></span>
</div>
<script type="text/javascript">
function onSelect(date){
	$('#result').text(date)
}
</script>
属性 说明 默认值
validParams 默认值为YMD.表示日期显示成YYYY-MM-DD YMD 如果是其它格式则不强制检查. 如配置成YM再实现formatter可实现例1效果2019-07-05
